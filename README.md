# Polar Flow – Personal Workout Data Export

Dieses Repository enthält meine aus **Polar Flow** exportierten Trainings- und Aktivitätsdaten.  
Die Daten dienen der **Analyse, Visualisierung und als realer Beispieldatensatz** für Software-, Daten- und KI-Experimente.

---

## Inhalt

Die Daten stammen aus dem offiziellen **Polar Flow Datenexport** und umfassen u. a.:

- Sporteinheiten / Workouts
- Trainingsdauer, Distanz, Kalorien
- Herzfrequenzdaten (sofern enthalten)
- Zeitstempel und Sportarten
- Aktivitäts- und Trainingszusammenfassungen
